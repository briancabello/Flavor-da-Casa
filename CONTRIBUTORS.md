Flavor da Casa was built collaboratively as an NBCC coursework project (Winter 2026), developed through Jira-managed sprints with a `main -> sprint -> feature` Git branching strategy and pull-request review between both contributors.

## Working Agreement

Both contributors worked across the full stack. The split above reflects primary ownership areas, not a strict frontend/backend division; both reviewed and contributed to each other's modules through pull requests during sprint cycles.
